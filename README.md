# 🚑 Emergency Resource Allocation System

A fully interactive AI-powered Emergency Resource Allocation System built using HTML, CSS, JavaScript, Leaflet.js, and OpenStreetMap. This project demonstrates how Artificial Intelligence search algorithms can be applied to real-world emergency healthcare scenarios for efficient resource allocation, ambulance dispatching, and route optimization.

## Features

🚨 **Emergency Response Simulation**

* Interactive accident site detection
* Hospital and ambulance station mapping
* Road network and junction visualization
* Real-time emergency resource allocation
* Intelligent patient routing system

🧠 **AI Search Algorithms**

* Breadth-First Search (BFS) for nearest hospital discovery
* Depth-First Search (DFS) for emergency area exploration
* Best-First Search for priority-based patient allocation
* A* Search for optimal emergency route planning
* Algorithm comparison and visualization

🏥 **Healthcare Resource Management**

* ICU bed availability monitoring
* Hospital resource tracking
* Ambulance station management
* Emergency patient prioritization
* Resource utilization analysis

🗺️ **Interactive Map Visualization**

* Leaflet.js powered map interface
* Dynamic route highlighting
* Color-coded algorithm paths
* Hospital, ambulance, and accident markers
* Zoomable and interactive emergency network

📊 **Analytics & Monitoring**

* Real-time system statistics dashboard
* Hospital and ambulance availability overview
* Route distance calculations
* Path comparison results
* Algorithm performance visualization

🎯 **Patient Priority Queue**

* Critical patient management
* Serious, moderate, and stable patient categorization
* Priority-based resource assignment
* Emergency treatment routing

🎨 **Modern User Interface**

* Professional emergency control dashboard
* Responsive and mobile-friendly design
* Interactive result panels
* Animated route rendering
* Clean dark-themed interface

## Technical Highlights

This project demonstrates the practical application of Artificial Intelligence search techniques in emergency healthcare management. It implements graph-based modeling of hospitals, ambulance stations, accident locations, and road junctions, enabling intelligent decision-making through BFS, DFS, Best-First Search, and A* Search algorithms.

The system dynamically calculates routes, evaluates healthcare resources, prioritizes emergency cases, and visualizes search paths on an interactive map. By combining heuristic search, graph traversal, pathfinding algorithms, and real-time visualization, the project showcases how AI can optimize emergency response operations and support critical healthcare decision-making.

A powerful project for learning Artificial Intelligence, graph algorithms, pathfinding techniques, heuristic search, emergency management systems, and interactive web application development.
